## 一些有意思的 C++ 练习项目

注意：查看 PPM 文件需要使用相关插件或查看器，不可直接查看其二进制文件

可以通过 ppm/bin/ffmpeg 下的 MP4 视频简要查看效果

包括以下内容：
1. 基于 ppm 的计算机图形学学习源码与成品
2. Ragine 静态图形生成引擎 - Shader Version
3. Ragine 静态图形生成引擎 - Ray Tracing Version （保留了传统着色方式，需要调用 legend API）

使用开源程序：
1. stb_image.h : (GitHub) Sean Barrett - nothings/stb (https://github.com/nothings/stb/blob/master/stb_image.h)
